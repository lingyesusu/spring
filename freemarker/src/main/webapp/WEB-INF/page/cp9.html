1,macro:宏指令

  • 栗子1：无参数的macro
    • <#macro test> 我是无参数的macro <@test/>
  • 栗子2：有参数的macro
    • <#macro test param1 param2> 我有参数的macro,param1=${param1},param2=${param2} <@test param1="java" param2="python"/>
  • 栗子3：有默认参数的macro
    • <#macro test param1 param2="python"> 我有参数的macro,param1=${param1},param2=${param2} <@test param1="java" param2="hello python"/>
  • 栗子4：有多个参数的macro
    • <#macro test param1 param2="python" paramExt...> 我有参数的macro,param1=${param1},param2=${param2}
    ${paramExt['param3']} ${paramExt['param4']} <@test param1="java" param2="hello python" param3="javascript" param4="nodejs"/>

2,nested

    <#macro test param1="java"> ${param1}
    <#nested param1,"我的nested参数">
  • 调用
    • <@test param1="java";loopVar1,loopVar2> hello${loopVar1},${loopVar2}
     <@test param1="python";loopVar1> hello${loopVar1}

3,函数

    <#function doAdd param1 param2> <#return param1+param2>
  • 调用
    • 你好，我是调用${doAdd(100,100)}
